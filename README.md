# Moodle REST Plugin

This project is just a test to create a simple plugin for Moodle that can response to a REST request.

## Setup

Clone Moodle and checkout the Moodle version you like

```bash
git clone https://github.com/moodle/moodle.git
cd moodle
git checkout v4.3.0
cd ..
```

To start moodle run:

```bash
docker compose up -d
```
